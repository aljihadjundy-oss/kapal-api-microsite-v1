Kebijakan Privasi.

Ringkasan bagaimana Sahabat Kapal Api mengumpulkan, memakai, dan menjaga data kamu saat mendaftar jadi member.

1. Pendahuluan

Saat kamu mendaftar lewat form "Ambil tempat duduk" di beranda, kami mengumpulkan data berikut:

  • Nama panggilan
  • Nomor WhatsApp
  • Kota domisili
  • Minat / fokus kamu saat ini (opsional, dari pilihan yang tersedia di form)

2. Tujuan Penggunaan Data

Data yang kamu berikan dipakai semata untuk mengirim informasi agenda dan program komunitas Sahabat Kapal Api lewat WhatsApp — misalnya pengingat acara, undangan workshop, atau kabar program baru. Data minat & kota membantu kami mengirim info yang relevan buat kamu, bukan pesan pukul rata ke semua orang.

3. Penyimpanan & Keamanan Data

Data kamu disimpan pada sistem internal komunitas dan tidak dibagikan ke pihak ketiga mana pun tanpa izin, kecuali jika diwajibkan oleh hukum yang berlaku. Kami berupaya menjaga data ini dengan langkah keamanan yang wajar sesuai skala komunitas.

4. Hak Pengguna

Kamu berhak untuk:

  • Meminta salinan data yang kami simpan tentang kamu
  • Meminta data kamu diperbarui atau dihapus dari sistem kami
  • Berhenti menerima info agenda & program (opt-out) kapan saja

Semua permintaan di atas bisa diajukan langsung lewat WhatsApp, dan akan kami proses secepatnya.

5. Kontak

Ada pertanyaan soal privasi atau data kamu? Hubungi kami langsung lewat WhatsApp Sahabat Kapal Api.
